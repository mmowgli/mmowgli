Video Page

Mmowgli games have traditionally been run with the support of an associated portal site. When Mmowgli games have been run from the MOVES Institute at the Naval Postgraduate School, a page has been maintained on that portal which aggregates videos from previous games. This page has tradionally be located at https://portal.mmowgli.nps.edu/game-wiki/-/wiki/PlayerResources/Video+Resources.

This link can be changed by a game designer through the Game Designer->Customize game->Global Settings->Header & Footer Links->Videos Link menu item.
