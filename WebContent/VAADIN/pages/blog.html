Blog

Mmowgli games have traditionally been run with the support of an associated blogging site. This is external to Mmowgli, and is a place to keep players updated with trending game issues. The blog may be part of a larger game-specific portal which provides other services desired by the game designer.

When a blogging site has been prepared, the Mmowgli header and footer links which now display the page you are readingd can reference the blog. This link can be changed by a game designer through the Game Designer->Customize game->Global Settings->Header & Footer Links->Game Blog Link menu item.
